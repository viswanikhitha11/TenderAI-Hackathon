# TenderAI - Government Tender Evaluation Platform

AI-powered prototype for automated government tender eligibility screening.

## Problem Statement
Government tender screening is often manual, time-consuming, and prone to inconsistencies.

## Solution
TenderAI automatically extracts eligibility clauses from uploaded tender PDFs and evaluates bidder eligibility using AI-assisted rule logic.

## Features
- PDF tender upload
- Clause extraction
- Automatic eligibility evaluation
- Risk scoring
- Officer decision support
- Explainable AI reasoning

## Tech Stack
- Python
- Streamlit
- Pandas
- PDFPlumber

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Expected Impact
- 80% reduction in screening time
- improved transparency
- scalable deployment for government procurement
